Distributed Multi-Model Analytics for E-commerce Data

The project demonstrates how MongoDB, HBase, and Apache Spark can be combined to analyze large-scale e-commerce data.
It focuses on selecting the right data model and processing engine based on data characteristics and analytical needs.

Here are technologies used:
MongoDB,Apache HBase (time-series session data),Apache Spark (batch processing & Spark SQL)Docker,Faker, Pandas, NumPy

Dataset
Synthetic e-commerce data generated with Python: Users,Products,categories, User sessions and Transactions Covers browsing behavior, purchases, inventory, and revenue over 90 days.
Analytics Implemented,Product popularity and revenue analysis, User segmentation and cohort analysis, Conversion funnel analysis, Customer Lifetime Value estimation, Product affinity analysis.
